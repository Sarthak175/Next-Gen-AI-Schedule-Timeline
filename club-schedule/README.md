# Event Schedule - 5-Day Technical Fest

A pure HTML, CSS, and Vanilla JavaScript event schedule timeline with animated backgrounds, interactive tabs, and smooth scrolling effects.

## Features

- 🎨 Animated sparkle background
- 📅 5-day event schedule with tab navigation
- ⏱️ Auto-scroll to upcoming events
- 📊 Scroll progress indicator
- 🎯 Interactive event cards with hover effects
- 📱 Side panel for event details
- ✨ Gradient text animations
- 📱 Responsive design

## How to Use

Simply open `index.html` in any modern web browser. No build process or dependencies required!

## Project Structure

```
club-schedule/
├── index.html    # Main HTML file
├── style.css     # All styles and animations
└── script.js     # Event handling and DOM manipulation
```

## Technologies

- Pure HTML5
- Pure CSS3 (with animations)
- Vanilla JavaScript (ES6+)

No frameworks, no dependencies, no build tools needed!
